# Mobdiaa Activities Dataset Use Case

## Overview
The Mobdiaa Activities Dataset is a comprehensive collection of data related to women's entrepreneurship and small business activities under the Mobdiaa initiative in Abu Dhabi. The dataset includes detailed information about various activities categorized into creative industries, retail services, and more. It is available in both English and Arabic, making it accessible to a diverse audience.

## Features
- **Multilingual Support**: Data available in English and Arabic.
- **Detailed Categorization**: Includes activity IDs, English and Arabic names, and categories.
- **Annual Updates**: Ensures data remains relevant and up-to-date.

## Key Use Cases
1. **Policy Development**: Support policymakers in identifying gaps and opportunities in women's entrepreneurship programs.
2. **Academic Research**: Aid researchers in studying gender dynamics in business.
3. **Community Awareness**: Inform the public about diverse women-led entrepreneurial activities.
4. **Business Opportunities**: Help entrepreneurs explore opportunities within the Mobdiaa initiative.
5. **NGO Initiatives**: Assist NGOs in creating targeted programs for women's empowerment.

## Dataset Details
- **Format**: XLSX
- **Language**: English, Arabic
- **Spatial Coverage**: Abu Dhabi, UAE
- **Temporal Coverage**: Ongoing

## Prerequisites
1. Python (version 3.7 or higher)
2. Pandas library (`pip install pandas`)
3. Requests library (`pip install requests`)
4. Internet connection to download the dataset

## How to Use
1. Clone the repository:
    bash
    git clone https://github.com/your-repository/mobdiaa-dataset-use-case.git
    
2. Navigate to the project directory:
    bash
    cd mobdiaa-dataset-use-case
    
3. Run the script to download, load, and analyze the dataset:
    bash
    python mobdiaa_analysis.py
    

### Output
- The dataset will be downloaded and saved as `Mobdiaa_Activities_Dataset.xlsx` in the project directory.
- Basic information and the first five rows of the dataset will be displayed.
